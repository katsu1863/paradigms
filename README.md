# Paradigms
- Each project runs independently of each other
- For Project 4, run `./build.bash` for Linux systems and `./build.bat` for Windows

# Controls
- Arrow keys to move
- Spacebar to shoot boomerangs
- 'e' to edit map
- 'a' to add sprites - mouse click to place them
- 'r' to remove spirtes - mouse click on sprite to remove
- 'c' to clear map
- 'l' to load map
- 's' to save map
- 'q' or 'esc' to quit game
- Editing capabilities are not implemented in Project 6
